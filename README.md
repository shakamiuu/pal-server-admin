#### 暂定功能
| 命令                    | 功能                   |
|-----------------------|----------------------|
| /Shutdown {秒} {消息}    | 服务器在秒数后关闭,将收到您的消息的通知 |
| /DoExit               | 强制停止服务器              |
| /Broadcast {消息}       | 向服务器中的所有玩家发送消息       |
| /KickPlayer {SteamID} | 将玩家从服务器上踢出           |
| /BanPlayer {SteamID}  | 从服务器封禁玩家             |
| /ShowPlayers          | 显示所有已连接玩家的信息         |
| /Info                 | 显示服务器信息              |
| /Save                 | 保存数据存档               |
