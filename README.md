#### 暂定功能

|     | 服务器部署     | 功能                                       |
| --- | -------------- | ------------------------------------------ |
| ✓   | 服务器状态监控 | 实时显示服务器运行状态                     |
| ✓   | 一键部署       | 一键部署游戏服务端                         |
| ✓   | 重启服务器     | 重启服务端释放内存                         |
| ✓   | 更新游戏       | 更新服务端游戏版本                         |
| ✓   | 修改游戏设定   | 直接修改游戏设定，不用登录服务器           |
|     | 存档备份       | 将游戏存档保存到本地                       |
|     | 存档恢复       | 本地存档恢复到服务器                       |
|     | 定时任务       | 配置定时执行重新或更新任务                 |
|     | 自动释放内存   | 监控服务器内存状态自动公告游戏内玩家并重启 |

#### 安装教程

1. 下载最新安装程序或免安装包
2. 安装或解压
3. 运行`幻兽帕鲁GM工具.exe`
